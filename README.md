## プロジェクト概要

MyPageにアクセスし、日記をつけることが出来るアプリケーションです。

## 使い方

・トップページでログインまたはサインアップができます。<br /><br />
・MyPageに入ると日記をつけることが出来ます。各項目を入力して書き込むボタンを押すと日記が書き込まれます。<br /><br />
・過去の日記が右に表示されるので、過去の日記を閲覧したり、削除することが出来ます。
